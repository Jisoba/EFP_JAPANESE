////////////////////////////////////////////////////////////////////
EFP4.0 日本語化MOD
////////////////////////////////////////////////////////////////////

◆概要
S.T.A.L.K.E.R. Anomaly EFP4.0 Hotfix6.1用の日本語化MODです。

100 Rads Bar(JP) で配布されている日本語化MODをベースにしています。
https://discord.com/invite/VqSNxrReVn

本MODはSTALKER-Anomaly-modded-exesのパッチを適用して、独自のexeファイルを作成しています。
これによりマルチバイトフォント環境下での改行バグを修正しています。
STALKER-Anomaly-modded-exes
https://github.com/themrdemonized/STALKER-Anomaly-modded-exes

◆注意事項
本MODは独自のexeファイルを作成している都合上、予期しない動作をする可能性があります。
また出力されたログファイルが文字化けする場合があります。

◆導入手順
1. EFP4.0をインストール(Hotfix適用含む)
2. 本MODのbinディレクトリ配下のexeファイルで、EFP4.0インストール先のbinディレクトリ配下の同名ファイルを置き換える。
3. 下記Anomaly本体の日本語化MODを導入
   https://github.com/Jisoba/STALKER_ANOMALY_JAPANESE_LINE_FEED_BUG_FIX/releases
4. Mod Organizer 2にてAnomaly本体の日本語化MODの優先度を最低(MODリストの一番上に配置)にする。
5. Mod Organizer 2にて本MODの優先度を最優先(MODリストの一番下に配置)にして、ゲームを起動してください。
